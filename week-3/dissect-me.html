Release 1: Bones

If a web page is a body, then HTML are the bones. HTML provides the structure and form for your page, and if you see an element outside of the HTML tag, something is probably very wrong.

Now that you have the page set up, it's time to build a skeleton. The first driver should open the file in Sublime, type "html" and hit the tab key. Sublime should automatically add a basic HTML skeleton. With your pair, discuss the following:

  • What is an <html> tag?
  • What is a <head> tag?
  • What information should be in the <head> tag?
  • What purpose does the <title> tag accomplish?
  • What information should live in the <body> tag?
